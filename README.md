# Role-Based Access Control (RBAC)

## Project Description
This project demonstrates Authentication and Role-Based Access Control (RBAC) using Node.js, Express.js, MongoDB, JWT, and bcrypt.

## Features
- User Registration
- User Login
- JWT Authentication
- Role-Based Authorization (Admin/User)
- Protected Routes

## Technologies Used
- Node.js
- Express.js
- MongoDB
- JWT
- bcryptjs
- Postman

## API Endpoints
- POST /api/auth/register
- POST /api/auth/login
- GET /api/auth/admin
- GET /api/auth/profile

## How to Run
1. Open the backend folder.
2. Run `npm install`.
3. Start the server using `node server.js`.
4. Test the APIs using Postman.
